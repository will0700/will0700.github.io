Projects

Menthol

Python | Django | SQLite | Bootstrap | HTML5 | CSS | JavaScript

Menthol is a budget and expense tracking application to mimic popular budgeting app Mint. Users will create entries for their bank accounts and credit cards, and expense categories with set budgets. They will then log their expense transactions to track spending and allocate them to different expense categories so that the application tracks how much has been spent and how much is left in the budget.

AutConnect - WIP

Java | Spring Boot | Spring Web Security | MySQL | JSP/HTML | CSS | JavaScript

AutConnect was developed as a tool for therapists in behavior therapy working with autistic children. Web Security is used to create users with three privilege levels: supervisors, therapists, and parents. Supervisors are able to register new clients and assign therapists. Therapists will log a daily note for each session, which is then viewed by supervisors and parents to track progress of the clients' therapy sessions.
